About

Yelp Help uses probabilistic programming to discover "tipping points" in restaurant reviews where user satisfaction changes from positive to negative. It then learns to identify comments, grammatical patterns, and entity-evoked-sentiments that occur at tipping points so that we can predict the coming fall in fortunes before it happens.
